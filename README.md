# VidGenie

Video Generator

### Install deps for GStreamer

```
$ sudo pacman -U https://archive.archlinux.org/packages/g/gst-plugins-good/gst-plugins-good-1.20.5-4-x86_64.pkg.tar.zst
$ sudo pacman -U https://archive.archlinux.org/packages/g/gst-plugins-good/gst-plugins-ugly-1.20.5-4-x86_64.pkg.tar.zst
$ sudo pacman -U https://archive.archlinux.org/packages/g/gst-plugins-ugly/gst-plugins-ugly-1.20.5-4-x86_64.pkg.tar.zst
```
